
# Student Data Analyzer
This python mini project proccess raw student marksheet data and generate clean academic marksheets report 

**It Proccess Operation likes**
 - avg marks of each student
 - avg class marks
 - failed student list
 - toppers and compartment students list
 - 3 top most and lowest marks of class

 ## project structure
 **data\\** - contain the raw marks of students  
 **notebooks\\** - contain jupyter notebook file that operate in flow and presents result  
 **reports\\** - this contains the report/outcome files after operations  
 **src\\** - this is a self-build package , define different modules in it for working with specific tasks

 ## How to run the project
 git clone https://github.com/Sumit-Maulekhi/student_data_analyzer.git   
 cd student_data_analyzer

 ## open the notebook
 notebooks/main.ipynb
